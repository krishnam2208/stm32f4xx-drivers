# STM32F4xx Driver Development

This repository contains bare-metal driver development for the STM32F407VG microcontroller using Embedded C.

## Features Implemented

* GPIO Driver
* GPIO Initialization
* GPIO Read/Write APIs
* GPIO Clock Control
* Register-Level Programming

## Target MCU

* STM32F407VGTx

## Tools Used

* STM32CubeIDE
* Embedded C
* Git & GitHub
* ST-Link Debugger

## Future Development

* RCC Driver
* SPI Driver
* I2C Driver
* UART Driver
* Interrupt Handling (EXTI/NVIC)
